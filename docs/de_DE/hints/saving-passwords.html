Password speichern

WARNUNG: Sie haben die Einstellung 'Passwort speichern' gewählt. Dies führt dazu, dass Ihr Passwort im Klartext in Ihrem Home-Verzeichnis unter *nix beziehungsweise in Ihrem Benutzerprofil unter Windows gespeichert wird. Wenn Sie das nicht möchten drücken Sie Abbrechen.

pgAdmin verwendet den 'pgpass' Mechanismus von PostgreSQL. Unter *nix wird das Passwort in ~/.pgpass gespeichert und unter Windows wird es in %APPDATA%\PostgreSQL\pgpass.conf (%APPDATA% sind die Anwendungsdaten Ihres Benutzerprofils) gespeichert. Der Mechanismus wird standardmäßig für alle Anwendungen verwendet die Bibliothek libpq nutzen um auf den Server zuzugreifen. Dies beinhaltet Konsolen-Anwendungen wie pg_dump und pg_restore, andere GUI-Anwendungen und Treiber wie zum Beispiel psqlODBC. Das bedeutet, dass diese Anwendungen sich unter Verwendung des gespeicherten Passwortes automatisch mit dem Server verbinden können. Wenn Sie dies nicht möchten, sollten Sie die Option Passwort speichern von pgAdmin nicht verwenden.

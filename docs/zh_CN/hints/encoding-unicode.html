Win32服务器之上的Unicode编码

数 据 库 使 用 Unicode（UTF8） 编 码 来 保 存 和 操 作 运 行 于 Win32 服 务 器 之 上 的 PostgreSQL 8.0 服 务 器 数 据。

因 为 操 作 系 统 没 有 提 供 足 够 的 支 持，Win32 之 下 的 PostgreSQL 8.0 无 法 依 据 适 当 的 次 序 对 字 符 串 进 行 排 序。

针 对 当 前 安 装， 推 荐 使 用 本 地 化 编 码、 在 不 同 操 作 系 统 之 上 运 行 或 者 将 数 据 库 升 级 到 有 专 门 代 码 可 以 支 持 Unicode 排 序 的 PostgreSQL 8.1。

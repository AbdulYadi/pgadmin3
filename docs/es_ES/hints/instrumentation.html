Instrumentación del servidor

El servidor carece de funciones de instrumentación.

pgAdmin III usa alguna funciones de soporte que no están disponibles por defecto en todas las versiones de PostgreSQL. Éstas habilitan algunas tareas que hacen la vida más fácil al tratar con archivos de registros (log) y de configuración..

Al instalar PostgreSQL 8.0 o superior usando el instalador Windows, sólo necesita marcar la opción "admin option".

Al compilar desde los fuentes, los archivos necesarios pueden ser encontrados en el subdirectorio xtra del árbol de fuentes de pgAdmin. Para PostgreSQL 8.0, copie el directorio admin al directorio contrib de los fuentes de postgresql, y ejecute make y make install desde allí. Para PostgreSQL 8.1, use el directorio admin81 para ello. Después que el módulo está instalado, necesita crear las funciones de instrumentación en su base de datos de mantenimiento usando el script admin.sql (admin81.sql para PostgreSQL 8.1) que están usualmente ubicados en el directorio share de pgsql (p.e. /usr/local/pgsql/share)

La instrumentación extendida de servidor no está soportada para PostgreSQL 7.3 y 7.4.

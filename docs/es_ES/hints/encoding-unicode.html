Codificación Unicode en un servidor Win32

La base de datos es creada para usar codificación Unicode (UTF8) para almacenar datos y es operada en un servidor PostgreSQL 8.0 coriendo en un servidor Win32.

La implementación Win32 de PostgreSQL 8.0 no es capaz de ordenar cadenas de caracteres de acuerdo a secuencias de ordenamiento correctas, debido a que el sistema operativo no ofrece suficiente soporte para ello.

Para esta instalación, se recomienda usar una codificación especifica de su localización, correr el cluster en un sistema operativo diferente o actualizar su cluster de base de datos a PostgreSQL 8.1 que tiene código adicional para resolver este problema de ordenamiento unicode.

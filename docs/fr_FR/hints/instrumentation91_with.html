Outils serveur

Le serveur ne dispose pas des outils supplémentaires.

pgAdmin III utilise quelques fonctions de support qui ne sont pas disponibles par défaut pour toutes les versions de PostgreSQL. Ces fonctions activent des fonctionnalités qui rendent la tâche plus simple pour traiter des journaux applicatifs et les fichiers de configuration.

L'adminpack est installé et activé par défaut si vous avez utilisé l'installeur one-click de PostgreSQL. Sur Unix, vous pouvez installer le module contrib, soit avec l'outil d'installation soit par compilation.

Une fois que l'extension est installée, vous devez cliquer sur le bouton « Fix it! », pour que pgAdmin installe cette extension dans la base de données de maintenance.
